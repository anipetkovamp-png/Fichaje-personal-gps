# Fichaje-personal-gps
App de control de fichaje horario diseñada para dispositivos móviles. Permite registrar entradas, salidas y anotaciones diarias con geolocalización GPS. Organiza tu jornada laboral de forma eficiente, visualiza totales mensuales y genera resúmenes para compartir. Gestión de turnos y datos totalmente privada en tu navegador
